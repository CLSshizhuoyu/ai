# 中文

**AI**是一个开源项目，提供了生成文本、图片和视频的工具，包括高质量视频生成。
	该项目设计简单高效，基于Python，易于使用。

- **文本生成**：根据给定的提示生成创意且连贯的文本。
- **图片生成**：从文本描述中创建独特图片。
- **视频生成**：根据文本提示生成视频。
- **高质量视频生成**：根据文本描述生成高分辨率、细节丰富的视频。
